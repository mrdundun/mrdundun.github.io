Hexo+Github搭建静态博客

发表于 2016-04-22   |   分类于 博客运营

安装node.js

1.安装Homebrew

/usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"`

HomeBrew 安装命令地址

2.安装node.js

brew install node

安装hexo

1.安装hexo

npm install hexo-cli -g

2.安装hexo-developer-git

npm install hexo-deployer-git --save

3.创建blog目录

4.初始化hexo

cd到blog目录后执行

hexo init blog
npm install

5.启动hexo服务

hexo s

6.本地预览

http://localhost:4000

关联github

1.在github项目中找到https地址

https://github.com/xxx/xxx.github.io.git

2.修改blog目录中的_config.yml

deploy:
    type: git
    repository: https://github.com/xxx/xxx.github.io.git
    branch: master

冒号后要有一个空格

3.生成ssh key

进入~/.ssh/目录

执行命令

ssh-keygen -t rsa -C "test@qq.com"`

生成id_rsa.pub文件

4.在github填入ssh key信息

Github账户 –> Settings –> SSH keys –> add SSH key

将id_rsa.pub文件中的内容复制并保存

发布文章

1.生成文章

hexo new “postName”
名为postName.md的文件会建在目录/blog/source/_posts下

2.clean

cd到blog目录后执行

hexo clean

3.生成

cd到blog目录后执行

hexo g

4.发布

cd到blog目录后执行

hexo d

安装theme

1.下载theme

cd到 blog 目录下执行如下命令：

git clone https://github.com/iissnan/hexo-theme-next themes/next

2.配置theme

将blog目录下_config.yml里theme的名称landscape修改为next

3.更改theme内容

比如修改名称，描述，头像。
修改blog/_config.yml文件和blog/themes/next/_config.yml文件中对应的属性名称即可。

NexT 使用文档

绑定域名

1.添加CNAME文件

在/blog/themes/landscape/source目录下新建文件名为：CNAME文件，直接将域名写入。

2.万网解析域名

记录类型：CNAME

记录值：xxx.github.io.（不要忘了最后的”.”号）
