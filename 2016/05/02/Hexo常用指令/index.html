Hexo常用指令

发表于 2016-05-02   |   分类于 博客运营

指令

init

$ hexo init [folder]

新建一个网站。如果没有设置 folder ，Hexo 默认在目前的文件夹建立网站。

server

$ hexo server

启动服务器。默认情况下，访问网址为：http://localhost:4000/。

选项 描述
-p, –port 重设端口
-s, –static 只使用静态文件
-l, –log 启动日记记录，使用覆盖记录格式

New

$ hexo new [layout] <title>

新建一篇文章。如果没有设置 layout 的话，默认使用_config.yml中的 default_layout 参数代替。如果标题包含空格的话，请使用引号括起来。

generate(或g)

$ hexo generate` 或 `$ hexo g

生成静态文件。

选项 描述
-d, –deploy 文件生成后立即部署网站
-w, –watch 文件生成后立即部署网站

deploy(或d)

$ hexo deploy

部署网站。

选项 描述
-g, –generate 部署之前预先生成静态文件

clean

$ hexo clean

清除缓存文件 (db.json) 和已生成的静态文件 (public)。
