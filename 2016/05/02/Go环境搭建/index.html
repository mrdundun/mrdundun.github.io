Go环境搭建

发表于 2016-05-02   |   分类于 Go

安装Go

安装Go开发包

brew install go

设置环境变量

vi ~/.bash_profile

### 打开编辑.bash_profile文件

export GOPATH=$HOME/go

export PATH=$HOME/bin:$GOPATH/bin:$PATH

### 添加环境变量

source ~/.bash_profile

### 保存并生效

测试运行

package main

import "fmt"

func main() {

    fmt.Printf("Hello, world")

}

上述代码保存成一个test.go文件，输入命令go run test.go来测试运行。

输出结果：

Hello, world

配置IDE

安装GoSublime

通过Package Control安装GoSublime

设置环境变量

Sublime——》Preferences——》Package Settings——》GoSublime——》Settings - Default

打开文件后将env改为如下内容

"env": { "GOPATH":"$HOME/go","PATH": "$HOME/bin:$GOPATH/bin:$PATH" },

测试效果

Ctrl+B快捷键，输入: 

go run test.go

希望得到的结果是输出Hello, world，但是报错

Abort. Cannot start MarGo: Exception: Cannot find command `C:\Users\xxx\AppData\Roaming\Sublime Text 3\Packages\User\GoSublime\windows-x64\bin\gosublime.margo_r16.05.01-1_go1.6.1.exe`

运行以下命令安装margo来修正：

go get -u github.com/slene/margo

margo被放置在目录：

GOPATH/bin/
