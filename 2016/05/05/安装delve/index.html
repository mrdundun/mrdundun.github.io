安装delve

发表于 2016-05-05   |   分类于 Go

确认安装Command line tool

安装

终端中输入以下命令：

xcode-select --install

在弹出的对话框中点击安装

验证

终端中输入以下命令：

xcode-select --install

出现以下提示表示安装成功：

error: command line tools are already installed, use "Software Update" to install updates

创建自签名证书并安装

  1. 打开Keychain Access
  2. Keychain Access菜单——》证书助理——》创建证书

  3. 选择一个名字，之后按图所示进行设置

  4. 设置证书有效期（1年到10年）

  5. 一路“继续“，直到出现指定证书位置对话框，按照下图设置。如果没法将证书存到“系统”下，那么先创建到“登录“下，之后再导入到“系统”下。

  6. 完成上述操作后，会看到新的证书已经安放到“系统”下。

  7. 右键点击证书——》显示简介，出现如下对话框：

    点击信任——》使用此证书时——》始终信任，如下图：

  8. 设置app访问证书权限，如下图

![image](8_访问权限控制.png)

  1. 重启电脑，一定要重启，否则会在调试时出现出现以下问题：

    delve could not launch process: could not fork/exec

  2. 创建文件夹

    $GOPATH/src/github.com/derekparker

  3. 进入刚刚创建的文件夹，用git下载delve，执行下面命令：

    git clone git@github.com:derekparker/delve.git

在Visual Studio Code下调试
